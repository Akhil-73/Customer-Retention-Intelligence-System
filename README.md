# Customer Retention Intelligence System

## Project Overview

The **Customer Retention Intelligence System** is an end-to-end data analytics project designed to help businesses understand customer behavior, identify churn risks, and improve customer retention through data-driven insights.

This project simulates a real-world business environment by integrating SQL, Python, Excel, and Power BI into a complete analytics workflow.

---

## Objectives

- Analyze customer behavior and engagement
- Identify factors contributing to customer churn
- Build customer retention metrics
- Generate business insights using SQL and Python
- Create interactive dashboards in Excel and Power BI
- Support business decision-making through analytics
